# WOFOST Yield Gap Analysis — Project Structure

This project runs the WOFOST 7.3 crop simulation model over multiple fields and farms
(DR and ZW regions, 2023–2025) to simulate potential yield (Yp) and water-limited yield
(Ywl), compare them against measured farm yields, and analyse the resulting yield gaps
and farm performance.

## Core Modeling Files

### `modeling/main.py` — Main Simulation Pipeline

The entry point that runs the full modelling workflow end-to-end. Call with:

```bash
python modeling/main.py
```

**What it does (in order):**

1. **Extract agro-management data** (`extract_agro_management_data`)
   - Reads `crop_registration_basic_*.xlsx` files for every farm from
     `RAW_DATA_PATH/1_crop_management_data/`
   - Normalises crop names (see `CROP_NAME_MAPPING`), resolves WOFOST variety names
     (see `resolve_variety` / `_POTATO_WOFOST_VARIETIES`)
   - Loads irrigation flags from matching `crop_registration_extra_*.xlsx` files and
     schedules timed irrigation events (20 mm every 20 days, efficiency 0.8)
   - Writes one `agro_<field_id>.yaml` per field to `input/agro/`

2. **Extract soil data** (`extract_soil_site_data`)
   - Reads `general_soil_characteristics.xlsx` (columns: Lutum → C, Dichtheid → D,
     Silt → S, OS → OM)
   - Applies Wösten pedotransfer functions (`PedotransferFunctionsWosten`) to derive
     Van Genuchten parameters
   - Converts them to WOFOST soil parameters via
     `ClassicalSoilWaterBalanceParameterProvider`
   - Validates physical constraints (ordering SM0 > SMFCF > SMW, etc.)
   - Writes one `soil_<field_id>.yaml` per field to `input/soil/`

3. **Build field-to-location map** (`get_ID_field_to_location_map`)
   - Reads `locations_data.xlsx`, parses DMS coordinate strings (e.g.
     `52°39'29.2"N`) to decimal degrees

4. **Run WOFOST 7.3 for every field** (`run_model`)
   - Fetches NASA POWER weather data at field coordinates
   - Loads site, soil, crop, and agro-management parameters
   - Applies custom sugar beet parameters from `input/Wofost73_PP_sugarbeet.yaml`
     (`_apply_custom_sugarbeet_params`)
   - Runs **Wofost73_PP** (potential production) and **Wofost73_WLP_CWB**
     (water-limited production)
   - Retains the harvest-date row per year per crop; assigns crop name and harvest
     year by matching simulation days to campaign date ranges
   - Returns a merged DataFrame of PP + WLP outputs per field

5. **Save results**
   - `output/model_results/WOFOST73_PP_results.xlsx` — potential yield outputs
   - `output/model_results/WOFOST73_WLP_CWB_results.xlsx` — water-limited yield outputs

### `modeling/soil_param_calculation/pedotransfer_functions.py` — Wösten Pedotransfer Functions

Class `PedotransferFunctionsWosten` calculates Van Genuchten hydraulic parameters
(α, n, k_sat, θ_r, θ_s) from soil texture inputs:

| Input | Unit | Description |
|---|---|---|
| C | % | Clay content |
| D | g cm⁻³ | Bulk density |
| S | % | Silt content |
| OM | % | Organic matter |

Used by `modeling/main.py` in `extract_soil_site_data`.

---

### `modeling/soil_param_calculation/van_genuchten.py` — Van Genuchten Soil Water Equations

Class `ClassicalSoilWaterBalanceParameterProvider` converts Van Genuchten parameters
to WOFOST soil water balance parameters:

| Output | Description |
|---|---|
| SM0 | Soil moisture at saturation |
| SMFCF | Soil moisture at field capacity (pF 2.0) |
| SMW | Soil moisture at wilting point (pF 4.2) |
| CRAIRC | Critical air content for root growth (fixed: 0.04) |
| RDMSOL | Maximum rootable soil depth (fixed: 120 cm) |

Used by `modeling/main.py` in `extract_soil_site_data`.

## Running the Pipeline

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Run the full simulation (generates agro/soil YAMLs + model outputs)
python modeling/main.py

# 3. Run analysis (after main.py has completed)
python analysis/merge_crop_records.py
python analysis/filter_crop_records.py
python analysis/combine_crop_registration_extra.py
python analysis/lme_farm_performance.py
python analysis/enrich_best_worst_with_extra.py

# 4. Generate plots
python plotting/plot_wofost_analysis.py
python plotting/plot_yield_gap_combined.py
python plotting/plot_field_farm_performance.py
python plotting/plot_weather_nasa_power.py
```
